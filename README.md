#  Fake Interview Simulator

A fully frontend web application that simulates a real interview environment using timed questions, voice input, and confidence scoring logic.

---

## Features

-  Timed interview questions (60 seconds per question)
-  Voice input using Web Speech API
-  Confidence score based on response behavior
-  Real interview pressure simulation
-  Clean and professional UI
-  Fully frontend (No backend required)

---

## Technologies Used

- HTML
- CSS
- JavaScript
- Web Speech API
